# 🌍 MiniMundo

> Una aplicación móvil híbrida educativa diseñada para el aprendizaje temprano de niños de 3 a 5 años.

![Estado](https://img.shields.io/badge/Estado-En_Iniciación-brightgreen)
![Progreso](https://img.shields.io/badge/Progreso-10%25-orange)

**MiniMundo** utiliza tecnologías web modernas para ofrecer una experiencia interactiva, colorida y educativa, libre de publicidad, en un entorno completamente seguro para los más pequeños.

## ✨ Características Principales

- **Juegos Educativos**: Actividades interactivas para aprender letras, números, colores y formas geométricas.
- **Sistema de Recompensas**: Refuerzo positivo mediante la obtención de "estrellas" al completar actividades.
- **Panel para Padres/Docentes**: Seguimiento en tiempo real del progreso de aprendizaje y estadísticas de uso de los niños.
- **Experiencia Sensorial**: Uso de sonidos y audios para guiar el aprendizaje y mantener la atención.
- **Entorno Seguro**: Navegación protegida sin enlaces externos, sin compras in-app y sin anuncios.

## 🛠️ Stack Tecnológico

El proyecto está diseñado bajo una arquitectura Cliente-Servidor:

* **Frontend (App Móvil)**: React.js, Tailwind CSS (para interfaces amigables y accesibles), Capacitor (para compilar la aplicación a Android nativo).
* **Backend (API REST)**: Node.js, Express.js.
* **Base de Datos**: MySQL, gestionada a través de Sequelize (ORM).
* **Diseño UI/UX**: Figma.

## 🚀 Requisitos e Instalación

Para ejecutar este proyecto en un entorno local, se requiere:
- **Node.js** (v18 o superior)
- **MySQL Server** (v8.0 o superior)
- **Android Studio** (para el despliegue de la app mediante Capacitor en un emulador o dispositivo real)

*(Las instrucciones detalladas para levantar el entorno de desarrollo se añadirán conforme avance la implementación del código base).*

## 📅 Hoja de Ruta (Roadmap)

El desarrollo está organizado en Sprints de 2 semanas. Los hitos más importantes son:
1. **Hito 1**: Prototipo funcional en navegador web.
2. **Hito 2**: API REST funcional con persistencia de datos en MySQL.
3. **Hito 3**: Primera compilación nativa en Android vía Capacitor.
4. **Hito 4**: Integración total del sistema de recompensas y pruebas con usuarios.
