# Plan de Desarrollo - MiniMundo

## Cronograma de 11 Semanas

| Fase | Actividad | Tiempo |
| :--- | :--- | :--- |
| **1** | Investigación, análisis y diseño de la base de datos MySQL | 1 Semana |
| **2** | Diseño de UI/UX en Figma (Enfoque preescolar) | 2 Semanas |
| **3** | Configuración de Backend (Express + MySQL) y Modelado | 1 Semana |
| **4** | Desarrollo Frontend (React) - Juegos de letras y números | 2 Semanas |
| **5** | Desarrollo Frontend - Colores, figuras y sonidos | 1 Semana |
| **6** | Integración de Capacitor y funcionalidades táctiles | 1 Semana |
| **7** | Pruebas de usabilidad con niños y corrección de bugs | 2 Semanas |
| **8** | Documentación final y despliegue (APK) | 1 Semana |

## Hitos Principales
1. **Hito 1**: Prototipo funcional en navegador (Semana 4).
2. **Hito 2**: API REST funcional con persistencia de datos (Semana 5).
3. **Hito 3**: Primera compilación nativa en Android vía Capacitor (Semana 8).
4. **Hito 4**: Sistema de recompensas y progreso terminado (Semana 10).

## Metodología
Se implementará **SCRUM** con Sprints de 2 semanas de duración, realizando reuniones de seguimiento para evaluar el avance de los módulos educativos.
